AnimationsAnimations

On this page we have an rotation animation at arrow of a dropdown box. When you open the page, there is another wipe animation at the dropdown box.

transform: rotate(0.1deg); transition: all 0.5s;

We have added a slight animation to the navigation bar when you hover over it.

transition: background-color .3s;

We have the same effect on each button as on the navigation bar. Among other things we let the button get bigger or smaller. Test it yourself.

transition: background-color .3s; transform: scale(1.03); transform: scale(0.98);
